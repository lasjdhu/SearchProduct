# SearchProduct mobile App

## March 2023

## Usage
+ Virtual emulator or physical device connected
```
$ git clone git@github.com:re-roll/SearchProduct.git
$ cd SearchProduct
$ npm i
$ npm start metro
[PRESS "a" for Android]
```
+ Using Expo scan this QR: https://expo.dev/@jbkdfkl/searchproduct
